# 圖片資源說明

請將以下圖片放在此目錄中：

## 必需圖片：

- `logo.png` - Logo 圖片 (建議 60x60px)
- `icon-home.png` - 首頁圖標
- `icon-games.png` - 遊戲庫圖標
- `icon-news.png` - 新聞圖標
- `icon-community.png` - 社群圖標
- `icon-settings.png` - 設定圖標
- `game1.png` - 遊戲1圖片
- `game2.png` - 遊戲2圖片
- `game3.png` - 遊戲3圖片

## 圖片大小建議：

- Logo: 60x60px
- 圖標: 24x24px
- 遊戲圖片: 300x200px (卡片) 或 120x120px (列表項)

## 快速生成方案：

可以使用在線工具生成占位符圖片：

- Placeholder.com
- Picsum.photos
- Via.placeholder.com
