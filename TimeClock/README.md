# Da Ka - 打卡

一款輕量、離線、無依賴的時間打卡網頁工具。點擊即可記錄時間，勾選項目自動計算總時長，所有資料儲存在瀏覽器本地，重新整理／關閉頁面不遺失。

## 功能特色

- **一鍵打卡** — 按一下「打卡」按鈕，立即記錄當前時間（`YYYY/M/D H:MM:SS am/pm`）
- **勾選計時** — 勾選任意兩筆以上時間，自動計算最早到最晚的總時長（X 小時 Y 分 Z 秒）
- **現在時間** — 即時顯示目前時間（`YYYY 年 M 月 D 日 H:MM:SS am/pm`），每秒自動更新
- **匯出多格式** — 下拉選單可匯出為 CSV / TXT / MD，每種格式皆包含總時數行
- **清除全部** — 一鍵清空所有打卡記錄（含確認提示）
- **資料持久化** — 使用瀏覽器 `localStorage` 儲存，關閉頁面、重新整理資料不遺失
- **行動裝置支援** — 響應式設計，手機可以直接使用

## 快速開始

1. 下載 `index.html`
2. 用瀏覽器直接開啟（雙擊檔案或拖進瀏覽器）
3. 點擊 **🕒 打卡** 開始記錄

> 不需安裝任何軟體、不需伺服器、不需網路連線（僅首次載入需要）。

## 使用說明

| 操作 | 說明 |
|---|---|
| 點擊「🕒 打卡」 | 新增一條時間記錄 |
| 勾選記錄前方的 ☐ | 選取要計算的時間區間 |
| 點擊「匯出 ▼」→ CSV/TXT/MD | 下載打卡資料，底部附總時數 |
| 點擊「清除全部」 | 清空所有記錄（需確認） |

- **總時數計算規則**：從勾選項目中找出**最早**與**最晚**的時間，計算兩者之間的時長。
- 勾選不足兩筆時，會提示「請勾選兩筆以上時間以計算總時數」。

## 技術說明

- **純前端**：單一 `index.html`，無任何後端、無第三方函式庫或框架
- **儲存方式**：`localStorage`（金鑰：`times`）
- **時間格式**：
  - 打卡列表：`YYYY/M/D H:MM:SS am/pm`
  - 底部即時時間：`YYYY 年 M 月 D 日 H:MM:SS am/pm`
- **匯出格式**：
  - **CSV**：UTF-8 BOM 格式，可直接用 Excel 開啟
  - **TXT**：純文字清單
  - **MD**：Markdown 表格
- **支援瀏覽器**：Chrome、Edge、Firefox、Safari（含手機版）

## 部署到 GitHub Pages

1. 建立 GitHub 存放庫
2. 將 `index.html` 推送至 `main`（或 `gh-pages`）分支
3. 至 Settings → Pages → 選擇來源為 `main` → `/ (root)`
4. 幾分鐘後即可在 `https://<你的帳號>.github.io/<存放庫名稱>/` 存取
